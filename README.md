## Hi there, I'm Valentin!

I'm an **Engineering Informatics** student at **THM (Technische Hochschule Mittelhessen)**.
I'm passionate about bridging the gap between hardware and software, focusing on Embedded Systems, Real-Time Operating Systems, and clean Software Architecture.

---

### What I do

* **Embedded Engineering:** Programming microcontrollers (ESP32) with C++ and FreeRTOS.
* **Software Design:** Applying object-oriented patterns and clean architecture concepts.
* **Web Development:** Building frontends (React, TypeScript) and backends (Vert.x, Java).
